## Entrega Primer Parcial – Proyecto de Diseño de Software

Este repositorio corresponde al proyecto del **Equipo 4**, cuyo objetivo es desarrollar una app para registrar hábitos saludables (agua, sueño, ejercicio, etc.).  
Aquí se documenta y organiza la entrega de las **Fases 1 a 3**:

- Fase 1: Planificación del equipo, cronograma en Monday, Git Flow y roles
- Fase 2: Análisis del sistema, actores, funcionalidades y requisitos
- Fase 3: Arquitectura adoptada (MVC) y su justificación

Documento completo: [`docs/PROYECTO_HABITOS_EQUIPO_4.pdf`](docs/PROYECTO_HABITOS_EQUIPO_4.pdf)
